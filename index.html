Scalable Data Ingestion and Migration from Oracle to Azure using IICS and ADF

Designed and managed a scalable data ingestion framework using IICS to replicate data from Oracle to ADLS Gen2. Configured secure agents, enabled high availability with agent groups, and developed mass ingestion tasks for large data volumes. Implemented CDC for incremental loads, scheduled workflows, and ensured smooth migration from IICS to ADF. Maintained sync between dev and prod environments and modeled target layer objects for analytics.

Azure Data Factory - ETL

Built a scalable, metadata-driven data ingestion framework using Azure Data Factory and Databricks. Ingested terabytes of data from on-premises Oracle databases and flat files (CSV) into Azure Data Lake Storage Gen2. Leveraged parameterized pipelines to dynamically handle multiple sources, improving reusability and reducing maintenance. Ensured performance and reliability through efficient data partitioning and transformation using Databricks.

Localise Vehciles ETL

Developed an automated pipeline to ingest vehicle data from SFTP, CSV files, and APIs across multiple dealerships using Azure Functions and Azure Data Factory. Data is loaded into Azure SQL Database, where it undergoes validation and transformation. Validated data is then published to Google Merchant Center. Implemented robust logging for error tracking and built a Power BI dashboard for monitoring ETL health and generating business insights.
